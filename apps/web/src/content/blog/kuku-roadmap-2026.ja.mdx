皆さんの大きな関心と応援のおかげで、Kuku Editor が [Product Hunt で #2](https://www.producthunt.com/products/kuku) に届きました。自分のニーズを満たすための個人プロジェクトとして始まったものが、透明なコードとプライバシー優先という約束に支えられて、ずっと大きなものへと育ってきました。

## 気づきの瞬間

Product Hunt でのローンチ時、自信のある機能を揃えていました。双方向の \[\[wikilinks\]\] を備えた Tauri ベースのローカルエディター、Gemini を使った AI エージェント、Whisper によるローカル音声認識、そしてグラフビュー。すべて皆さんの Mac でローカルに動きます。

けれど反応は機能だけの話ではありませんでした。コメント、アップボート、そこから広がる会話 — それらすべてが、もっと深いところを指していました。人々は単に新しいノートアプリに興奮していたのではなく、**主導権** に飢えていたのです。

- _"コードをオープンソースにしますか？"_
- _"自分の AI キーを使えますか？"_
- _"プライバシーはどう検証すればいいですか？"_

メッセージは明確でした。ユーザーは、また別のプラットフォームに縛られたくない。信頼でき、検査でき、主導権を握れる道具を求めています。

そこで、決めました。

## 決断：完全な透明性

クライアントだけをオープンソースにするのではありません。**すべて** を公開します。

- **デスクトップクライアント**: MIT ライセンス。使い、フォークし、その上で作ってください。
- **サーバーインフラ**: AGPL ライセンス。同期サーバー、AI 処理バックエンド、データベーススキーマ、デプロイ設定まで、すべて GitHub に公開します。

ブラックボックスはなし。独占的なロックインもなし。データがシステムをどう流れるかを正確に監査でき、自分のインフラで動かすこともでき、改善をコミュニティに還元することもできます。

けれどオープンソースだけでは足りません。土台そのものも作り直しています。

## リビルド：ゼロから

あの機能群をリリースしたあと、間違った土台の上に積み上げていたと気づきました。React と Tiptap は MVP には十分でしたが、本当に拡張可能なプラットフォームを作るには、もっと深く掘り下げる必要がありました。

エディターのコア全体を **SolidJS** と **純粋な ProseMirror** に移しています。なぜでしょうか？

- 純粋な ProseMirror は独自ラッパーがないことを意味します — 誰もがライセンスの心配なくエディターを拡張できます
- SolidJS の細粒度なリアクティビティが、必要としていたパフォーマンスを与えてくれます
- きれいな土台の上でだけ、長く望んできたプラグインアーキテクチャが可能になります

ゼロからの全面書き直しです。時間はかかりますが、これから作るすべてにとって正しい土台になります。

## 三つの道

ここから面白くなります。一本の道を強制したりしません。選ぶのはあなたです。

### 1. すべて自分でホスティング

サーバーコードを自分のハードウェアにデプロイしてください。ホームサーバー、NAS、月 5 ドルの VPS でも動かせます。Docker Compose ファイルとデプロイガイドを用意しています。データは、あなたが管理するインフラから出ていきません。

### 2. 自分のサービスをつなぐ (BYO)

すでに信頼しているクラウドサービスがありますか？自分の S3 バケット、OpenAI の API キー、あるいはローカル LLM (Ollama) をつないでください。Kuku は、すでに使っているサービスの上に乗るインターフェース層になります。

### 3. 便利さを買う

サーバー管理はしたくない？ [kuku.mom](https://kuku.mom) — マネージドサービスをご利用ください。インフラ、アップデート、バックアップを私たちが担当します。便利さに対してお金を払うことになりますが、コード自体はセルフホスティングできるものと同一です。

**そして約束します**: データを人質には取りません。今日 kuku.mom で始めて、明日セルフホスティングに移っても構いません。いつでも、すべてを書き出せます。

## これからの計画

### 1.0 オープンソースローンチ

SolidJS と ProseMirror による完全リライトで、最初のオープンソースリリースを迎えます。含まれるもの:

- **MeetingNote**: ローカル Whisper の書き起こしを統合した講義録音
- **GitHub Sync**: ボールトを GitHub リポジトリに同期する無料のバージョン管理
- **Homebrew 配布**: Mac ユーザー向けのワンコマンドインストール
- **サーバーコード一式**: セルフホスティングガイドと Docker 構成

### 検索とグラフの強化

ローカルの埋め込みモデルによる意味理解:

- **ローカル埋め込み**: Ollama や ONNX Runtime を使い、埋め込みモデル (nomic-embed-text など) を完全にオンデバイスで実行。API 呼び出しもインターネット接続も不要です。
- **ハイブリッド検索**: キーワード検索 (BM25) と意味検索 (ベクトル類似度) を組み合わせ、単語一致ではなく意味でノートを見つけます。
- **スマートグラフビュー**: 明示的なリンクだけでなく、意味の類似度でノート同士のつながりを可視化。つながっているとは知らなかった関連ノートを発見できます。
- **文脈に応じた提案**: 書いている最中、いま開いているドキュメントの意味内容に基づいて関連ノートを提案します。

すべてローカルで動きます。ノートは、あなたのマシンから離れません。

### 同期とモバイル

端末をまたぐ同期と、モバイルコンパニオン:

- **ゼロ知識同期**: kuku.mom、セルフホスティングサーバー、あるいは自分の S3 バケットを通した端末間の E2E 暗号化同期。サーバーは読むことのできない暗号化 blob だけを保管します。
- **モバイルアプリ**: 移動中の素早いキャプチャと閲覧のためのネイティブ iOS・Android コンパニオン。モバイル端末でも完全な編集が可能です。
- **競合解決**: 複数端末で同じノートを編集しても、賢くマージします。
- **オフラインファースト**: すべてオフラインで動作します。接続時にバックグラウンドで同期が走ります。

### 拡張エコシステム

コミュニティの革新に向けたプラットフォーム開放:

- **Plugin SDK**: 拡張を作るための公開 API。フル文書と TypeScript 定義付き。
- **プラグインマーケットプレイス**: Kuku の中からそのままコミュニティプラグインを見つけてインストール。
- **リアルタイム協業**: プラグインシステムのおかげで、共有ワークスペース向けの Y.js ベース協業編集などの拡張が可能に。
- **Web Clipper**: どんなウェブサイトからでも内容を取り込むブラウザ拡張。
- **AI メモリ共有**: 他の AI アシスタントが、安全なローカル API を通じてあなたのナレッジベースにアクセス・参照できるようにします。ノートが、あなたの使うあらゆる AI ツールの長期記憶になります。

## 変わらない価値

これらの核となる原則を守りながら進みます:

- **ローカルファースト** — ファイルはあなたのものです。プレーンな .md、いつでもアクセス可能、決してロックインされない。
- **極端なプライバシー** — 強制的なアカウント作成なし、許可のないデータ収集なし、明示的なオプトインなしのテレメトリーなし。
- **透明なエコシステム** — すべてのコードは GitHub にあります。オープンソースリリースのあとは、プラグインとモバイル同期で広げていきます。
- **選択の自由** — 私たちのクラウドを使う、セルフホスティングする、自分のサービスをつなぐ。道具は私たちが作りますが、主導権はあなたに渡します。
